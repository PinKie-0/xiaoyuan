# 快速开始测试

## 方法一：自动化测试（推荐）

### 步骤 1：运行测试数据初始化

```bash
python init_test_data.py
```

### 步骤 2：运行测试

```bash
# 使用测试运行器
python run_tests.py -q
```

## 方法二：手动测试

### 步骤 1：初始化测试数据

```bash
python init_test_data.py
```

### 步骤 2：启动应用

```bash
python run.py
```

### 步骤 3：打开浏览器

访问：`http://localhost:5000`

### 步骤 4：按照测试文档执行测试

参考 `docs/软件测试文档-按实际功能整理.md` 中的测试用例进行测试。

## 常见问题

### Q: 端口被占用怎么办？

A: 修改 `run.py` 中的端口号，或关闭占用 5000 端口的程序。

### Q: 数据库错误怎么办？

A: 删除 `instance/xiaoyuan.db` 后重新运行 `python init_db.py`。

### Q: 忘记密码怎么办？

A: 运行 `python reset_user_passwords.py` 重置所有测试账号密码为 Test123456。

### Q: 如何重置测试数据？

A: 删除数据库文件，然后重新运行：
```bash
python init_db.py
python init_test_data.py
```
